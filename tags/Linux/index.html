Centos7防火墙设置 查看已开放的端口 1firewall-cmd --list-ports 开放端口（开放后需要要重启防火墙才生效） 1firewall-cmd --zone=public --add-port=3338/tcp --permanent 关闭端口（关闭后需要要重启防火墙才生效） 1firewall-cmd --zone=public --remove-port=3338/tcp --permanent 重启防火墙 1firewall-cmd --reload 开机启动防火墙 1systemctl enable firewalld 开启防火墙 1systemctl start firewalld 禁止防火墙开机启动 1systemctl disable firewalld 停止防火墙 1systemctl stop firewalld 联系博主 微信 公众号
